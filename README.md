### 🧭 About

I build systems at the intersection of **robotics**, **computer vision**, and **modern AI** — from multispectral crop/weed segmentation in the field, to 3D retrieval with CLIP + FAISS, to diffusion-based game content and imitation learning for Mobile ALOHA. I enjoy turning research prototypes into things that actually run.

### 🚀 Featured Projects

| Project | Description |
| --- | --- |
| [**3D object retrieval tutorial**](https://github.com/inkyusa/3D_object_retrieval) | Toy example for 3D model retrieval from text or image using CLIP embeddings + FAISS indexing. |
| [**GenAI game scene generation**](https://inkyusa.itch.io/low-poly-ai-generated-mid-age) | Unity + 3D retrieval + diffusion + monocular depth estimation demo (playable sample). |
| [**deepNIR**](https://inkyusa.github.io/deepNIR_dataset/) | Synthetic NIR image generation + improved fruit detection with deep learning. |
| [**Mobile ALOHA (fork)**](https://github.com/inkyusa/act_vel_added) | Imitation learning + co-training for Mobile ALOHA — bug fixes, velocity feature, extra tooling. |
| [**weedNet**](https://github.com/inkyusa/weedNet) | Multispectral crop & weed dataset for semantic segmentation. |
| [**deepFruits**](https://www.kaggle.com/datasets/enddl22/deepnir-11fruits) | Kaggle-hosted 11-fruits bounding-box annotation dataset. |
| [**gym_rotor**](https://github.com/inkyusa/gym_rotor) | Foundational continuous-control RL examples for quadrotors. |
